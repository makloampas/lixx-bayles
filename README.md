## Setup

```bash
git clone https://github.com/makloampas/lixx-bayles
cd lixx-bayles
npm install
cp .env.example .env
# Edit .env sesuai kebutuhan
node index.js
```

## Konfigurasi
| Variable | Keterangan |
|---|---|
| PHONE_NUMBER | Nomor WA kamu (jika pakai Pairing Code) |
| MONGO_URI | URI MongoDB kamu |
